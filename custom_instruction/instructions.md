# USBank shield design system

1. this design system consists of only components in the ui/ directory and styles in the tailwind.config.ts ALWAYS use these before making something custom
2. Put the US-Bank-Logo.png in the top left corner of the app unless the user asks otherwise and size it appropriately
3. This component library doesnt really have a card component, dont try to wrap everything in a card. Its designed to look better without them
